## 🛠️ 开发工具推荐

作为一名 Web 开发者，选择合适的工具可以大大提升开发效率。今天分享一些我日常使用的工具。

### 代码编辑器

#### VS Code
目前最流行的代码编辑器之一，强大的插件生态让它几乎无所不能。

**推荐插件**：
- **Prettier** - 代码格式化
- **ESLint** - 代码质量检查
- **GitLens** - Git 增强工具
- **Auto Rename Tag** - 自动重命名配对标签

### 版本控制

#### Git + GitHub
代码版本管理的标配，GitHub 不仅是代码托管平台，更是开源社区的聚集地。

**常用命令**：
```bash
# 初始化仓库
git init

# 提交更改
git add .
git commit -m "feat: 添加新功能"

# 推送到远程
git push origin main
```

### 包管理器

#### pnpm
相比 npm 和 yarn，pnpm 更快、更节省磁盘空间。

```bash
# 安装依赖
pnpm install

# 添加包
pnpm add package-name

# 运行脚本
pnpm dev
```

### 浏览器开发工具

#### Chrome DevTools
强大的浏览器开发者工具，用于调试、性能分析、网络请求监控等。

**常用功能**：
- Elements - 查看和修改 DOM
- Console - JavaScript 调试
- Network - 网络请求分析
- Performance - 性能分析

### API 测试工具

#### Postman
API 开发和测试的必备工具，支持各种 HTTP 请求方法。

### 设计工具

#### Figma
基于浏览器的设计工具，团队协作非常方便。

### 部署平台

#### Vercel / Netlify
一键部署静态网站，支持自动构建和 CDN 加速。

#### Cloudflare Pages
免费且强大的静态网站托管服务。

### 学习资源

- **MDN Web Docs** - Web 开发权威文档
- **Stack Overflow** - 问题解答社区
- **GitHub** - 开源项目学习

### 总结

工具只是辅助，真正重要的是理解原理和持续学习。选择适合自己的工具，不断优化工作流程，才能成为更高效的开发者。
